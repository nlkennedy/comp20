Write Up

Website Citation: https://theysaidso.com/api/

API citation: https://quotes.rest/

The TheySaidSo API is used to generate famous quotes. They have multiple categories of quotes, such as inspirational or sports quotes. They have a quote of the day, and a quote of the day for each category.

I used the categories options for the API requests. Once a category is specified as a query string, the API will return the quote of the day from the specified category.

There are multiple uses for this API, especially since it changes the quotes every day so that new content can be seen. First, it could be used on a sports blog to display a different sports quote every day, using the "sports" category. Second, it could be used in a professional setting for a job search website, since they also have a "management" category.

A drawback of using this API (the free version), is that it allows a limited number of API calls per hour.
